# SRB2K Replay Organizer

A simple script that creates a neat directory structure for your replays. Useful when you want to save your replays automatically but not get lost afterwards.

![](gif.gif)

# Requirements

- Any recent (past 2-3 years, maybe?) version of Python 3

# Installation and usage

1. Race;
2. After having raced, and having saved some replays, run the script. Instructions on how to install and run it are located below.

## Windows

Copy `organizer.bat` and `rep-organizer.py` to your replay folder, then run `organizer.bat`:

You can also run the python script from your terminal:

```
path\to\replay\folder> python rep-organizer.py
```

## Linux, \*BSD, or MacOS

Copy `rep-organizer.py` to your replay folder, then run it from your terminal:

```bash
$ cd path/to/replay/folder/
$ python rep-organizer.py
```

You can also `chmod +x` it if you want.
